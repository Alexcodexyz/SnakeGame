# 🐍 贪吃蛇游戏

一个现代化的、基于HTML5 Canvas的贪吃蛇游戏，具有流畅的动画、直观的控制和丰富的自定义选项。

![游戏截图](game-screenshot.png)

## ✨ 特性

### 🎮 核心功能
- **流畅游戏体验** - 60fps稳定帧率，响应式控制
- **智能速度控制** - 可调节游戏速度（50-300），数值越大速度越快
- **自动加速机制** - 每吃5个食物自动提升游戏速度
- **高分记录系统** - 本地存储最高分，挑战个人记录

### 🎨 视觉设计
- **现代UI设计** - 深色主题配合渐变色彩
- **网格背景** - 半透明网格线，提升视觉体验
- **精致游戏元素** - 蛇身带边框，食物有层次感
- **流畅动画效果** - 模态框动画，按钮悬停效果

### 🎯 用户体验
- **直观控制** - 方向键控制，支持连续转向
- **清晰状态显示** - 实时得分、最高分、速度显示
- **友好提示** - 游戏结束弹窗提供选择自由
- **响应式设计** - 适配不同屏幕尺寸

## 🎮 游戏玩法

### 基本操作
- **↑ ↓ ← →** - 方向键控制蛇的移动
- **开始游戏** - 点击开始按钮或按任意方向键
- **暂停/继续** - 点击暂停按钮
- **重置游戏** - 点击重置按钮回到初始状态

### 游戏规则
1. 使用方向键控制蛇的移动方向
2. 吃到红色食物获得10分
3. 撞墙或撞到自己游戏结束
4. 每吃5个食物游戏速度会增加
5. 通过速度滑块可以调节基础移动速度

### 速度控制
- **滑块范围**: 50-300
- **速度逻辑**: 数值越大，移动速度越快
- **初始速度**: 50（最慢，300ms延迟）
- **最大速度**: 300（最快，50ms延迟）

## 🛠️ 技术实现

### 核心技术
- **HTML5 Canvas** - 高性能2D图形渲染
- **原生JavaScript** - 无依赖，轻量级
- **CSS3** - 现代化样式和动画
- **LocalStorage** - 本地数据持久化

### 代码结构
```
SnakeGame/
├── index.html      # 主页面结构
├── style.css       # 样式和动画
├── script.js       # 游戏核心逻辑
├── keyboard_test.html  # 键盘测试工具
├── speed_test.html     # 速度测试工具
├── test_report.md  # 测试报告
└── CHANGELOG.md    # 版本更新记录
```

### 关键算法
- **速度映射**: `delay = 350 - speed_value`
- **自动加速**: 每5个食物减少10ms延迟
- **碰撞检测**: 边界检测 + 自身碰撞检测
- **食物生成**: 随机位置，避免与蛇身重叠
